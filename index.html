Common questions

A 30% deposit is due before work starts — for example €90 on the €300 package — with the rest due once you've reviewed the finished site and you're happy with it. The exact split is something we'll confirm during our initial chat.
Yes — 100%. Once the final payment is made, all the files are yours. No licences, no monthly fees, no lock-in. I'll hand over the code and you can host it anywhere, or I'll keep it hosted for free on GitHub Pages.
Most basic sites are ready within 1–2 weeks from our first chat. Sites with extra features like booking systems or databases can take 2–4 weeks, depending on how much content you have ready and how quickly we can agree on a design.
For most small text or detail changes, it's easiest to just send me a message and I'll handle it — that's covered by the hourly maintenance rate. If you'd prefer to manage content yourself day to day, we can talk about setting that up too.
Hosting is free through GitHub Pages, so the only ongoing cost is your domain name, typically around €10–20 a year. I can register and set this up for you, or help you do it yourself if you'd rather own it directly.
Every site comes with a 3-month warranty after launch, so any bugs or issues from the original build are fixed free of charge. After that, fixes and updates are billed at the standard €25 per hour maintenance rate.
